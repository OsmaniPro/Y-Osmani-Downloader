# Youtube Downloader bot 

## Bot Ini Bisa Mendownload Dari Youtube Via telegram
## Prerequisite
    ffmpeg
  
    
## install dependencies
    pip3 install -r requirements.txt


## Setup Bot
    - Change configuration config.py  File
    - install dependencies
    - python3 -m bot
